条件判断

计算机之所以能做很多自动化的任务，因为它可以自己做条件判断。

#python

形式：
if <条件判断1>:
    <执行1>
elif <条件判断2>:
    <执行2>
elif <条件判断3>:
    <执行3>
else:
    <执行4>
    
##例子：
输入用户年龄，根据年龄打印不同的内容。
1.
age = 3
if age >= 18:
    print('your age is', age)
    print('adult')
else:
    print('your age is', age)
    print('teenager')

2.
age = 3
if age >= 18:
    print('adult')
elif age >= 6:
    print('teenager')
else:
    print('kid')
